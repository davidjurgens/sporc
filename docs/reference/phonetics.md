# Phonetics

Word-level alignment and formant measurement from source audio. Optional and
heavy — install with `pip install sporc[phonetics]` and an `ffmpeg` binary on
PATH. See the [Phonetics guide](../guides/phonetics.md) for the narrative
walkthrough.

::: sporc.phonetics
    options:
      show_root_heading: true
      members_order: source
      filters: ["!^_"]
